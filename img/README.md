# 图片资源目录说明

## 目录结构

- `zh/` - 中文界面截图和相关图片
- `en/` - 英文界面截图和相关图片

## 文件命名规范

- 使用描述性的文件名，避免使用随机字符串
- 推荐格式：功能描述_日期.png
- 例如：profit_calculation_20251205.png

## 图片使用说明

在文档中引用图片时，请使用相对路径：

```markdown
![利润计算界面](../img/zh/profit_calculation.png)
![Profit Calculation Interface](img/en/profit_calculation.png)
```
